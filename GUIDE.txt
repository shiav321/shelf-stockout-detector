Q: "Tell me about your most impactful project"

A: "I built an AI-powered Shelf Stock-Out Detection system for retail stores.
   
   The problem: Retail stores lose ₹50,000+ daily because staff manually check 
   shelves every 2 hours and miss empty spots, causing lost sales.
   
   My solution: I used OpenCV and TensorFlow with MobileNetV2 transfer learning 
   to analyze existing CCTV footage in real-time. The system divides each camera 
   frame into grid zones and classifies each zone as stocked or empty using texture 
   analysis and a trained CNN model. When more than 2 zones are empty, it instantly 
   alerts the store manager via WhatsApp.
   
   Results: 91% detection accuracy, alerts within 30 seconds of stock-out, 
   30% reduction in out-of-stock events, saving ₹15,000+ per store per day.
   
   Companies like Trax Retail built a $1 Billion business solving exactly this 
   problem — so I know the real-world demand exists."


Q: "What challenges did you face?"

A: "Three main challenges:
   1. Lighting variation — store lighting changes throughout the day affecting 
      image quality. I solved this with brightness augmentation during training.
   2. Limited data — I only had 200 shelf images. I used transfer learning with 
      MobileNetV2 pretrained on ImageNet, which gave me 91% accuracy with minimal data.
   3. Real-time performance — processing every frame was too slow. I optimized by 
      processing every 15th frame and using smaller 224x224 crops per zone."


Q: "How would you scale this?"

A: "Deploy the model on edge devices like NVIDIA Jetson at each store, 
   send alerts via MQTT to a central cloud dashboard on AWS. One dashboard 
   can monitor 1000+ stores simultaneously with <1 second alert latency."
